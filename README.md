# Python Student Database

## Description

The Python Student Database is a beginner-friendly project that demonstrates how to use dictionaries to store and manage student information. Each student record contains a name, age, course, marks, and grade. This project helps learners understand how dictionaries organize related data using key-value pairs.

## Features

- Store information for multiple students
- Display all student records
- Update student information
- Search for a student by name
- Calculate and display grades
- Practice working with dictionaries and loops

## Technologies Used

- Python 3

## Concepts Practiced

- Dictionaries
- Key-value pairs
- Functions
- Loops
- Conditional statements
- User input
- Lists (optional)

## Example Student Record

```python
student = {
    "name": "Samuel",
    "age": 22,
    "course": "Computer Science",
    "marks": 85,
    "grade": "A"
}
```
